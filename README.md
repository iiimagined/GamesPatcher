## 🧠 What this does

macOS has a built-in **Games app** — but it only shows apps that identify as games.

GamesPatcher lets you:
- Add anything to the Games library  
- Create clean launcher apps  
- Organize your setup properly  

No clutter. No weird hacks. Just works.

---

## ⚙️ How it works

macOS checks a simple value:
If it's set to **games**, it appears in the Games app.

GamesPatcher:
- Creates launcher apps with that value  
- Or patches existing apps safely
